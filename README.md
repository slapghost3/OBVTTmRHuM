# 手写数字识别系统 python1116

## 项目概述

手写数字识别系统是一个基于Python的项目，它集成了手写字识别、管理以及用户个人信息的维护等功能。

## 技术栈

- **后端语言**：Python 3.7
- **数据库**：MySQL
- **前端技术**：未明确说明
- **开发工具**：PyCharm

## 功能模块

- **手写字识别**：识别用户手写的数字
- **手写字管理**：管理已识别的手写数字记录
- **修改密码**：允许用户更改自己的密码
- **个人信息**：用户可维护自己的个人信息
- **用户管理**：管理系统中的用户账户

## 系统角色

系统包含以下角色：
- **普通用户**：可进行手写数字识别、管理个人信息等
- **管理员**：负责用户管理和系统维护

## 运行环境

- Python 3.7 或以上版本
- MySQL 数据库服务
- 支持前端的Web服务器

## 部署步骤

1. 环境准备：确保Python和MySQL的安装与配置
2. 数据库搭建：根据提供的数据库文件初始化MySQL数据库
3. 项目配置：配置项目所需的数据库连接等参数
4. 运行项目：启动后端服务及Web服务器
